Welcome to Moondash

Ever wanted a generic admin UI for REST endpoints, with the frontend moved out of the backend? Moonshot is a high-quality open source project with a ready-to-go application for managing information in your REST backends.

Simple

Just include our bundled JS and CSS into your page and configure Moondash to point at a compliant endpoint.

Sexy

Attractive, responsive theme and UX from the Moonbeam.

Modern

Built as an AngularJS single-page application with best practices.

Generic

Works with any server framework behind a REST endpoint.

Configurable

Easily adjust predefined settings such as the logo without changing Moondash code.

Extensible

Register new sections and views into the UX without touching Moondash code.

Quality

High-performance, testing, and documentation in our code means far less for you to sign up for.

Open

Fully open source project, in license, development, and attitude.
